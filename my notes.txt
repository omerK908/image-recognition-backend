create package.json - npm init -y
install nodemon - npm install nodemon --save-dev
create the js file you want: touch server.js
add to package.json the "start" under "scripts": 
"scripts": {
    ...
    "start": "nodemon server.js"
  }
now the nodemon will refresh the server after updates

install express: npm install express

install body parser for incoming requests: npm install body-parser

install bcrypt to hash passwords: npm install bcrypt-nodejs

connect to postgresql: npm install knex
                        npm install pg
